## 🆘 NEED HELP?

PROBLEM                          SOLUTION
──────────────────────────────  ─────────────────────────────────
MongoDB won't connect           Start mongod OR check Atlas settings
Admin login fails                Check spelling in .env.local
No data showing                  Wait 2 seconds, check MongoDB running
Port 3000 in use                 Use: npm run dev -- -p 3001
npm install errors              Delete node_modules, reinstall
